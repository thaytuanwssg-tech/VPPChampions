🔑 Anthropic API Key

Chatbot cần API Key để hoạt động. Lấy tại console.anthropic.com.

💡 Cài đặt tại: Netlify → Site configuration → Environment variables → thêm ANTHROPIC_API_KEY.

☁️ Kết nối Firebase

Dữ liệu lưu trên Google Firebase, dùng được mọi thiết bị.

📋 Hướng dẫn (5 bước):
1. Vào console.firebase.google.com → Đăng nhập Google
2. Add project → Đặt tên → Tạo
3. Firestore Database → Create database → Test mode → Done
4. Project Settings ⚙️ → General → Your apps → icon </> → Register app
5. Copy các giá trị và dán vào ô bên dưới
